- BOOTSTRAP -

Bootstrap es un framework que agiliza la programación mediante un conjunto de clases ya hechas.

Para utilizarlo hay que instalarlo primero en la pagina oficial y copiar las 2 nineas CDN para pegarlas en el "head", el script se puede poner al final del body para que las inimaciónes se cargen lo ultimo.

Contenedores


Existe "container" que deja un marjen horizontal y "container-fluid" que también deja un marjen horizontal pero mucho más pequeño

Tablas normal


Para hacer una tabla se pone "table"

Para darle un color se usa "table-*color*" que puede usarse tanto el el "table" como en los "tr"

También se puede usar "table-striped" para alternar colores grises, "table-hover" y "table-bordered" para un pequeño borde
