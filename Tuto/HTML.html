Textos

⬇️Etiquetas para modificar/usar textos⬇️

- "p" para un parrafo
- "pre" para hacer un parrafo que tenga en cuenta los espacios
- "i" para cursiva
- "em" como el "i"
- "mark" para subrallar
- "b" para negrita
- "u" Subralla
- "ins" para subrallar también
- Se puede usar "span" para agrupar un trozo de texto y modificarlo
- "del" para tachar
- Para bajar un caracter se usa "sub": H2O
- "br" para hacer un salto de linea 
- "hr" para hacer una linea como las de arriba
- "lorem(numero)" para crear textos aleatorios
- "small" para reducir el tamaño de la letra
- "big" para aumentar el tamaño de la letra

- "center" para centrar un texto
 

- Para citar a alguien en un texto se usa "blockquote", se le añade "cite" para decir de quien es, esto no se muestra, y para que se vea de quien es yo pongo un "footer"

  • Las "ul" utilizan signos para separar los puntos

    • Con "style="list-style-type: *tipo*"" se puede cambiar el formato de la lista
  • Elemento
- ol -
  1. Las "ol" utilizan un orden ascendente o descendente de elementos de la lista
  2. Elemento
    1. Con "type" se cambia el tipo de orden

Formularios

Para hacer un formulario se usa "form" a la que se le puede añadir "action", para decirle al servidor que le está mandando y "method", que puede ser "get", que se lo manda sin cifrar y "post" que si lo cifra al mandar los datos introducidos. El "name" dentro de un "input" es el nombre con el que se envia la información de ese input al servidor

Los formulariosse suelen meter en un "fieldset" que mete el formulario en una caja, y con la etiqueta "legend" (dentro del fieldset) se pone el nombre del formulario

Se usa "input" para meter un campo:
- El tipo de campo se define en "type".
- El texto mostrado en el imput se establece con "placeholder".
- El id para ser usado con un label se pone con "id".

Se usa "label" para poner un texto para un input y en el "for" se pone para que id.

Para enviar la información o para reiniciar los campos se usa un "button", con un "type" "submit" o "reset", tiene que estár dentro del form.

En los inputs de numeros se puede poner "min" "max"

Si los "label" están bien establecidos se podrá clicar en estos para referirte al "input".
