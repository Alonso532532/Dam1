Vinculación

Para vincular un archivo se tiene que poner en el head un "link" con un "rel="stylesheet"" y un "href" con la ubicación del archivo css

Esto también se puede usar para el icono de la pestaña, para este se usa lo mismo, pero en el "rel" se pone "shortcut icon"

Selección de elementos

Para indicar que todos los elementos de un mismo tipo tengan un formato especifico se pone: 
nav {
    background-color: rgb(116, 116, 116);
    padding: 17px;
    margin: 10px;
    border-radius: 10px;
}
Para indicar que los elementos con una clase tengan un formato especifico se pone: 
.cuadrado {
    background-color: rgb(76, 76, 76);
    padding: 15px;
    border-radius: 10px;
    margin-left: 20px;
    margin-right: 20px;
    color: rgb(240, 240, 240);
    font-size: 15px;
}
Para indicar que los elementos con una clase tengan un formato especifico se pone: 
#cuadrado {
    background-color: rgb(76, 76, 76);
    padding: 15px;
    border-radius: 10px;
    margin-left: 20px;
    margin-right: 20px;
    color: rgb(240, 240, 240);
    font-size: 15px;
}
Para indicar que los elementos dentro de un elemento tengan un formato especifico se pone: 
nav a {
    margin: 5px;
    color: rgb(0, 0, 0);
    text-decoration: none;
    background-color: rgb(91, 91, 91);
    padding: 10px;
    border-radius: 8px;
}
Para indicar que los elementos pegados a otro elemento dentro de ese otro elemento tengan un formato especifico se pone: 
nav > a {
    margin: 5px;
    color: rgb(0, 0, 0);
    text-decoration: none;
    background-color: rgb(91, 91, 91);
    padding: 10px;
    border-radius: 8px;
}
Para indicar que varios elementos tengan un formato especifico se pone: 
p,
pre,
div {
    font-family: Verdana, Geneva, Tahoma, sans-serif;
    margin-top: 5px;
}

Modificación de elementos

Para editar el texto: 
.pruebaT {
    color: rgb(240, 240, 240);
    font-size: 15px;
    font-weight: lighter;
}
