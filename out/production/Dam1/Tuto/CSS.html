Selección de elementos

Se le pueden asignar varias clases a un mismo objeto poniendo: class="clase1 clase2"
Para indicar que todos los elementos de un mismo tipo tengan un formato especifico se pone: 
nav {
    background-color: rgb(116, 116, 116);
    padding: 17px;
    margin: 10px;
    border-radius: 10px;
}
Para indicar que los elementos con una clase tengan un formato especifico se pone: 
.cuadrado {
    background-color: rgb(76, 76, 76);
    padding: 15px;
    border-radius: 10px;
    margin-left: 20px;
    margin-right: 20px;
    color: rgb(240, 240, 240);
    font-size: 15px;
}
Para indicar que los elementos con una clase tengan un formato especifico se pone: 
#cuadrado {
    background-color: rgb(76, 76, 76);
    padding: 15px;
    border-radius: 10px;
    margin-left: 20px;
    margin-right: 20px;
    color: rgb(240, 240, 240);
    font-size: 15px;

Para indicar que los elementos dentro de un elemento tengan un formato especifico se pone: 
nav a {
    margin: 5px;
    color: rgb(0, 0, 0);
    text-decoration: none;
    background-color: rgb(91, 91, 91);
    padding: 10px;
    border-radius: 8px;
}
Para indicar que los elementos pegados a otro elemento dentro de ese otro elemento tengan un formato especifico se pone: 
nav > a {
    margin: 5px;
    color: rgb(0, 0, 0);
    text-decoration: none;
    background-color: rgb(91, 91, 91);
    padding: 10px;
    border-radius: 8px;
}
Para indicar que varios elementos tengan un formato especifico se pone: 
p,
pre,
div {
    font-family: Verdana, Geneva, Tahoma, sans-serif;
    margin-top: 5px;
}
Para indicar que los elementos que estén directamente despues de otros tengan un formato especifico se pone: 
h2 + p{
    font-family: Verdana, Geneva, Tahoma, sans-serif;
    margin-top: 5px;
}
Para indicar que los elementos que estén despues de otros (da igual la distancia) tengan un formato especifico se pone: 
h2 ~ p{
    font-family: Verdana, Geneva, Tahoma, sans-serif;
    margin-top: 5px;
}
Se puede usar "*" para seleccionar todo lo que haya


Para referirte a que solo se le aplique el efecto cuando pase el cursor por encima: 
a:hover {
    background-color: rgb(81, 81, 81);
    padding: 9px;
    margin: 6px;
}
Para cuando este clicando: 
a:active {
    color: rgb(213, 213, 213);
}
Y para cuando se esté escribiendo en ese campo:

    background-color: red;

.pruebaTabla{
    background-color: black;
    display: inline-block;
}

.pruebaTabla td:nth-child(odd){
    background-color: rgb(88, 88, 88);
}

.pruebaTabla2{
    background-color: black;
    display: inline-block;
}

.pruebaTabla2 tr:nth-child(even){
    background-color: rgb(88, 88, 88);
}
"odd" para impares y "even" para pares y si se pone un numero solo se afectará esa posición
Si se usa "tr:nth-last-child(even)" aunque ponga even se selecciónan las pares

Posicionamiento (Display, Position, Grid y Flex)

Display

Hay tres tipos importantes: "display: inline;", "display: block;" y "display: inline-block;"

inline:
Si se pone “display: inline”, los elementos se mostrarán en estructura de linea, sin separación alguna y solo son textos por lo que no tienen ni altura ni anchura (Estos elementos no podrán tener formatos como width o heigh).





block:
Si se pone “display: block”, los elementos se mostrarán en estructura de bloque (con sus espacios y sus cosas).



inline-block:
Si se pone “display: inline-block”, los elementos se ponen en linea pero mantienen formato



"box-sizing: border-box;" sirve para que el conjunto de elementos que modifican el espacio que ocupa el elemento no supere el tamaño del elemento

Position

El "static" es el por defecto

El "relative" mueve el elemento desde su posición nativa

El "absolute" mueve el elemento desde su padre con posición distinta a estatica

El "Fixed" pega el elemento a la pantalla en la posición qe le digas, es el que está abajo a la derecha

"Float" permite definir de que forma se tiene que ordenar, este no es un position, pero lo pongo igualmente
